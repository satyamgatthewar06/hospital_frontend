## Quick Visual Reference

**When to use Media Queries:**
- Layout changes (sidebar, grid)
- Font size adjustments
- Padding/margin changes
- Element hiding/showing
- Grid column count changes

**Mobile-First Means:**
```
/* Base styles (mobile) */
.element { width: 100%; }

/* Enhance for tablet */
@media (min-width: 768px) {
  .element { width: 50%; }

  .element { width: 25%; }
}
```
